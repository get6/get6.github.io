今回は、企業活動において必ず押さえておくべき安全衛生とデジタルセキュリティについてお話しします。

## 産業災害と重大災害の定義

**産業災害**：労働者が業務遂行中に死亡したり、負傷・疾病にかかることを指します

**重大災害**：死亡者が1人以上発生するか、3か月以上の治療が必要な負傷者が2人以上、または負傷者（職業性疾病者）が同時に10人以上発生した場合です

## 事故発生の原因

産業災害事故は、労働者の不安全な行動と設備・環境の不安全な状態から発生します。

- **不安全な行動**：労働者が注意義務を怠ったり、危険な行動をする場合
- **不安全な状態**：設備や作業環境が適切に管理されておらず危険な状態になっている場合

不安全な行動を引き起こす原因としては、**慢心、焦り、思い込み、適当さ、無知、無能、無関心**があります。
特に、従業員50人未満の小規模事業所、入社1年未満の新規労働者、そして月曜日・金曜日の出勤後2時間以内に事故が多発する特徴があります。
中でも指の負傷が最も頻繁に発生するため、手の保護具の着用と注意喚起が重要です。

**災害防止の4原則**：

1. **予防可能の原則**：事故は事前に予防できるという原則です。
2. **損失偶然の原則**：損失が予測可能な範囲内で発生するよう管理する必要があります。
3. **原因連鎖の原則**：原因と結果が明確に結びつくよう分析する必要があります。
4. **対策選定の原則**：予防対策は合理的な基準に従って選択する必要があります。

**実践可能な予防対策**：

- 2人1組の作業でリスクを分散します。
- 整備・清掃・点検・補修作業時に、作業前後の安全ルールを徹底的に守ります。
- 新規労働者の教育と現場点検を強化します。

## ハインリッヒの事故発生理論

ハインリッヒは、事故が社会的環境と遺伝的要素、個人的欠陥、不安全な状態・行動、事故、災害へと連鎖的に発生すると説明しました。
また「**1:29:300の法則**」を通じて、大事故1件には29件の小事故と300件の些細な兆候が伴うとしました。
小さな兆候でも見逃すと大事故につながる可能性があるため、すべての警告をおろそかにしてはいけません。

|         | **ハインリッヒの災害予防5段階**       |
| ------- | ------------------------------------- |
| 第1段階 | 安全管理組織（Organization）          |
| 第2段階 | 事実の発見（Fact Finding）            |
| 第3段階 | 評価・分析（Analysis）                |
| 第4段階 | 是正策の選定（Selection of Remedy）   |
| 第5段階 | 是正策の適用（Application of Remedy） |

## デジタルセキュリティの重要性

安全事故に加えて、現代の企業にとって情報資産は核心的な競争力です。
個人情報漏洩事故の主な原因は、外部からのハッキングと内部からの漏洩です。

**情報保護10大ルール**：

1. 重要文書は機密等級・社外秘等級で区分します。
2. 機密文書は役員レベルの承認手続きを経る必要があります。
3. 文書の移動・コピー時には暗号化・アクセス制御を適用します。
4. 定期的なセキュリティ教育と点検を実施します。
5. 入退室管理とネットワークファイアウォールを徹底します。
6. パスワード管理・二段階認証など基本的なセキュリティルールを守ります。
7. バックアップと復旧計画を策定・点検します。
8. セキュリティ事故発生時には即座に対応プロセスを稼働します。
9. 外部協力会社・委託業者のセキュリティ管理も徹底します。
10. 最新のセキュリティ技術・ポリシーを継続的にアップデートします。

### 個人情報の定義と処理段階別の義務

個人情報とは、生存する個人を識別できるすべての情報を指します。
処理段階ごとに守るべき義務は以下の通りです。

1. **収集・利用（第15条）**：情報主体の同意、最小収集の原則を適用。
2. **提供（第17条・第18条）**：目的外の利用・提供の禁止、第三者提供時の手続き遵守。
3. **管理（第29条～第34条）**：安全措置義務、処理方針の作成、漏洩時の通知・届出要件。
4. **破棄（第21条）**：保有期間終了時に遅滞なく破棄する必要があります。

これに違反した場合、5千万ウォン以下の過料が課される可能性があります。

### CCTVと映像情報の処理

CCTVの設置・運用も「**映像情報処理機器運用・管理指針**（第25条）」に従う必要があります。
設置場所、目的、保管期間などを明確に告知し管理しなければなりません。
