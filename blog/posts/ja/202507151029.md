## 特許出願、こう準備しましょう

特許出願は複雑に見えますが、主要な手続きと注意事項さえ押さえれば、難しくなく進められます。

### 特許出願手続きの全体像

1. **相談**：発明届出書を作成して、特許事務所との相談を開始します。
2. **着手金の案内**：受任および前受金に関する案内を受けます。
3. **先行技術調査と登録可能性の検討**：既存に類似の特許があるか調査し、登録可能性を評価します。
4. **登録可能性の意見伝達**：調査結果に基づいた意見を受け取ります。
5. **明細書の草案作成**：特許明細書の草案を作成します。
6. **明細書の草案送付と確認**：草案を受け取り確認します。
7. **明細書の草案修正**：必要に応じて修正を依頼します。
8. **残金請求**：残金が請求されます。
9. **特許出願と出願報告**：最終明細書で特許を出願し、出願結果の報告を受けます。
10. **出願手続き終了と費用精算**：すべての手続きが完了し、費用が精算されます。

> **参考：** ほとんどの特許事務所は相談料を取らず、手数料で収入を得ています

### 国内・海外出願時の注意事項

- **海外出願の予定がある場合**：韓国出願日から1年以内に海外出願をすれば、韓国出願日を基準に遡及を受けられます。
- **実体審査と方式審査**：方式審査を通過すれば出願は完了し、実体審査で登録の可否が決まります。
- **特許の本質**：特許は技術を公開し、他者がそれを参考にできるようにする制度です。これにより技術の発展が促進されます。
- **特許検索**：類似特許を事前に検索する習慣が重要です。

## 財務・会計の基礎理解

### 企業のステークホルダーと財務情報

企業の財務情報はさまざまなステークホルダーにとって重要な役割を果たします。

| ステークホルダー | 主な関心事                           |
| ---------------- | ------------------------------------ |
| 投資家           | 投資価値、収益性                     |
| 債権者           | 利息および元金回収、キャッシュフロー |
| 消費者           | 企業の持続性、A/Sなど顧客管理        |
| 政府             | 税金徴収、企業支援および規制         |
| 従業員           | 収益性、成長性、適切な報酬           |
| 製品販売者       | 取引代金、成長性、販売戦略           |
| 原材料供給者     | 取引代金回収、収益性、生産戦略       |

### 企業活動と会計

- **企業活動**は、財務活動、投資活動、営業活動に区分されます。
- **会計**とは、企業のさまざまな取引（資産、負債、資本、収益、費用の変動）を一般企業会計基準に従って記録する作業です。

## 財務諸表分析、こう始めましょう

### 財務諸表の読み方

- **損益計算書**と**貸借対照表**を中心に財務比率を分析します。

### 財務比率分析とは？

- 財務諸表の項目間の数値を比較して、企業の財務状態と経営成果を把握する方法です。
- 主な比率：流動性、安定性、収益性、活動性など

### 標準比率の活用

- **自社の過去比率**：自社の過去の数値と比較
- **業界平均比率**：韓国銀行の企業経営分析、産業銀行の財務分析などで提供される業界平均と比較

## 税務の基本、必ず知っておくべきこと

### 税務申告と事業者登録

- **国税**は原則として納税者が自ら申告して税額が確定します。
- **事業者登録証**：法人は設立登記日、個人事業者は登録発行日が記載されます。

### 主な税金の種類

- 法人税、法人地方所得税、所得税（事業所得）、地方所得税、付加価値税など

### 税額減免制度

- 首都圏過密抑制圏域外で法人設立時に税額減免の恩恵
- 青年起業家であれば登録税・教育税の免除、所得税免除、取得税減免、財産税の恩恵など多様な支援があります。
- ただし、創業初期で売上が少ない場合、減免効果はそれほど大きくない可能性があります。

### 税務・ビジネス支援サービス

- **国税相談センター**：126番
- **ビジネス支援団**：1357番、ホームページ提供

### 適格証憑の受取規定

- 他の事業者に対価を支払う際は、必ず証憑書類（税金計算書、計算書、クレジットカード売上票、現金領収証など）を受け取る必要があります。
