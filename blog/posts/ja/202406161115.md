## 1講 どのように定義するか

ジェンダーとはどういう意味でしょうか？
ある人の生物学的性別と同じ意味でしょうか？ それとも社会的アイデンティティを理解する別の方法でしょうか？

![](assets/ebs-great-minds-4-1-1.webp)

ジェンダーは広範な論争を引き起こしてきた言葉でもあります。この言葉の解釈が主要な課題の一つです。世界の一部の地域ではジェンダーの概念に反感を示しています。一部の人々はそれが自国語で伝統的に男女の差異を規定する言語への挑戦だと考えています。

シモーヌ・ド・ボーヴォワール（Simone de Beauvoir）[1908-1986]は多くの人々の尊敬を集める20世紀の主要なフェミニスト哲学者です。『第二の性』（1949年に出版されたフェミニズム理論の古典）を著したフランスの作家でもあります。

> 女に生まれるのではない、女になるのだ。
> — Simone de Beauvoir

この宣言がセックスとジェンダーを区分する理論的基盤となりました。セックスは生物学的実体として理解される性別という概念であり、ジェンダーは特定の時代と場所で性別に付与された社会的意味を指す概念です。

生まれた時に与えられた生物学的性別が女性だからといって、女性らしく見えるように特定の方法で話したり歩いたりしなければならないことを意味するわけではありません。

私たちは結婚を社会制度と見なしています。結婚は社会が生物学的性別を組織する方式です。性別を基礎としていますが生物学に従ったものではありません。生物学によって決定される制度ではないのです。
愛する方法さえも社会が組織したジェンダー化された体の行為と見ることができます。
外見を表現すること、歩き方、話し方、仕事をすることさえも一種の社会的行為です。その一部は自ら決定し、一部は社会的・文化的規範に従って決定されます。

20世紀に、生物学的実体として理解される性別という概念と特定の社会で性別が担う社会的意味であるジェンダーの違いを明確に区別することが、さまざまな理由で重要になりました（同一賃金と同一労働など）。
生物学は女性が何をするかを決定しません。何をするかを決めるのは自分自身です。

性別とジェンダーの区分により女性に自由の地平を開きましたが、それは男性にも当てはまることでした。
男性も何が男性的で男らしいとはどういうことか、男になるとはどういう意味かについて確固たる考えを持って育ちます。こうした考えが強迫観念を生むことがあります。しかし男であるという観念は時代とともに変化してきました。

生まれた時に与えられた性別が、これからどう生きるかを決める要因ではありません。だからといって性別から完全に自由になるという意味でもありません。自由というのは常に特定の文化、特定の社会の中で生まれます。時によって自由であったりなかったりします。自由とは闘いです。自由は社会規範と慣習の制約を受けます。規範と慣習を変えようとする努力は時として非常に困難なものになり得ます。いつもジェンダーの変化は社会的圧力と伝統の圧力という制約の枠から逃れられません。

20世紀後半のフェミニスト理論で、シモーヌ・ド・ボーヴォワールの宣言はこのように解釈されました。
`性別は先天的カテゴリーだが、一方でジェンダーはその先天的カテゴリーを文化的・社会的に解釈する名称である。`

性別という概念もさまざまな方法（制度、文化、言語、医学的慣習、科学的慣習の歴史）で決定されてきました。
二分法とは何でしょうか？ 二つのうちどちらか一方にのみ属するという意味です。しかし性別にはインターセックスがあります。男性と女性の身体的特徴を両方持っています。こうした特性は動物にも見られます。

### 1講 講義まとめ

ジェンダー：英語の単語、論争の対象、馴染みのない言葉

- 性別（sex）：生物学的実体としての性
- ジェンダー（gender）：社会的・文化的意味としての性

ジェンダー理論の核心 -> 解釈（既存の用語/性別への挑戦）

フランスの実存主義思想家、シモーヌ・ド・ボーヴォワール

- 主体としての女性を強調した20世紀フェミニズムの先駆者
- 性別とジェンダーの区分の基盤を築いた。「女に生まれるのではない、女になるのだ」

生物学（性別）は人の人生を決定できない！
インターセックス（intersex）が存在する -> 男性と女性の二分法による身体規定に属さない人
ジェンダーの意味は自由に変えられるべき

1970年代の解釈：性別は先天的カテゴリー、ジェンダーは性別の社会的解釈
現代の解釈：性別の分類は歴史的流れ、言語、科学史によって異なる。性別を簡単に断定できない。

## 2講 選択ではなく現実

出産は女性の選択です。さまざまな方法で自分たちの自由を社会で表現することができます。女性の一部は女性に対する期待が強い社会に生きています。社会が女性の生物学的能力にこのような義務を課したために生まれました。

![](assets/ebs-great-minds-4-2-1.webp)

子供を産める生殖能力は女性が持つ生物学的能力の一つとして明示できます。しかし女性がどのような存在か、彼女の人生がどうあるべきかを規定してはなりません。
もし子供を産んでこそ真の女性であり、出産だけが真の女性の証だとすれば、それは社会規範を課すことです。
結婚せずに別の人生を望む女性がいますが、彼女のこうした望みがある社会やある集団では不適切な行為と見なされることもあります。
フェミニストは生物学に反対したり生物学的事実を否定したりしません。

性別とジェンダーを区別すべき理由は、性別だけで誰かの人生を予測してはならないからです。
トランスジェンダーは闘います。ジェンダーを変える医学的・法的自由のために。ジェンダーのカテゴリーが彼らの生きる現実を説明できるようにです。
性別とジェンダーの違いを知ることはフェミニストにとって非常に重要です。女性の自由を拡大し、男性優位主義に挑戦しました。男性が社会的に優位な位置を占めることが当然だという考えに対抗したのです。

シモーヌ・ド・ボーヴォワールの宣言により3つの論争が起こりました。

1. 女性のみを考慮した。
2. 男女のカテゴリーは常に異性愛の枠組みを前提としている。
3. 人間は性別間の物理的差異を指摘できるのか。

女性は自分たちの状況の中で自由であるためにもがかなければなりません。与えられた条件の中で状況を変えなければなりません。そして気づかなければなりません。いかなる自由も平等も、闘いによってのみ得られるということを。

### 2講 講義まとめ

生物学的性別で他者の人生を予測したり決定したりしてはならない
ジェンダーという現実は選択できない問題
与えられた性別では説明できないトランスジェンダー -> 新しいカテゴリーが必要な人生

生まれた時に与えられた性別

- 社会的現実
- 社会的アイデンティティ
- 自分の体に対する感覚
- 進路

ボーヴォワールの宣言に対する共通理解 -> 性別のカテゴリーは決まった事実ではなく状況
性別の違いは探そうとする時にのみ発見される -> 認識の枠組み
指定された性別は＜ジェンダー＞について決定できない

「女に生まれるのではない、女になるのだ」
-> 性別のカテゴリーは決まった事実ではなく状況。性別を理解する方法は時間と空間によって変化
-> 社会変革のプロセス、困難な交渉のプロセス

## 3講 言語の壁

シモーヌ・ド・ボーヴォワールは多くの尊敬を集める20世紀の主要なフェミニスト哲学者です。

![](assets/ebs-great-minds-4-3-1.webp)

シモーヌ・ド・ボーヴォワールが考慮しなかった一つの可能性はトランスジェンダーのアイデンティティです。
トランスジェンダーの存在は性別指定に亀裂を生みます。新しい言葉と新しいカテゴリーを主張するからです。
ジェンダーが置かれた制約と自由の間の矛盾を必ず理解しなければなりません。
私たちはあまりにも長い間、生物学的性別が社会的な生き方を意味すると考えてきました。

性別はその言葉自体を生み出す言語行為と分離できません。性別の言語的形態は社会的な性、ジェンダーの意味を持つと考えます。そしてジェンダーは外来語です。このジェンダーという意味を母国語に翻訳するのは非常に難しいです。
翻訳は必要ですが、翻訳の限界も知る必要があります。
反文化帝国主義の一部になる必要があります。文化帝国主義に反対しなければなりません。世界中の平等のためのフェミニストの努力を台無しにします。

### 3講 講義まとめ

シモーヌ・ド・ボーヴォワール『第二の性』

- 1949年に出版されたフェミニズム理論の古典
- 性別（生物学的性）とジェンダー（社会学的性）は同じだという通念に反旗、明確な違いを提示
- 性別は科学だけで決まる単純な事実ではなく、社会・文化の影響を受ける
- トランスジェンダーのアイデンティティを見落とした -> 男性、女性、そして新しいジェンダー -> 新しい言葉とカテゴリーを要求

他の言語における性別の翻訳問題

- 性別を表現する万国共通語はない
- 性別の文化的解釈 -> 言語によって異なるのは必然
- すべての時代と文化を包括する性別解釈はない

アメリカの文化的影響力、強大な英語の力
英語圏のジェンダー理論家たちの無意識的態度 -> ジェンダーを英語の単語ではなく万国共通語として認識
ジェンダー、フェミニズム理論の翻訳に対する英語圏の学者の無関心
ジェンダーが現在の概念になったのは20世紀半ば -> 性別のように言語ごとに意味が異なるジェンダー
翻訳の限界を知ることが重要
異なる言語間で同等の概念を確立することが重要
一つの言語がジェンダー用語を独占することはできない
単一言語（英語）への批判はグローバルなジェンダー運動に含まれる

## 4講 論争の中心

私たちはジェンダーが国際的に激しく議論される時代に生きています。
男女だけでなく多様なジェンダーがもっと生きやすい世界が来るよう努力すべきだと思います。
倫理的に、誰かが自分が特定のジェンダーに属すると言い、特定の代名詞で呼ばれることを望むという主張は無条件に尊重されるべきです。誰かが自分を定義する方法に疑問を投げかけてはなりません。自分を表現する自由を尊重しなければなりません。その中で得られる自尊心を否定してはなりません。

![](assets/ebs-great-minds-4-4-1.webp)

ある人々は既存のジェンダーカテゴリーから外れて生きることの方が良いと考え、また別の人々はジェンダーカテゴリーの中で認められながら人生の意味を見出します。生き方を一つに固定する必要はないと思います。
ある人々はジェンダーという用語にさまざまな社会的・経済的不安を結びつけます。多くの人の人生が破壊されたことをジェンダーという用語のせいにします。社会全般に不安定性をもたらすと非難します。ジェンダー検閲を支持する人々はジェンダーイデオロギーを共産主義に例えます。

すべての人にもう少し心を開いてほしいとお願いしました。他の人生も同等の価値を持つことを考慮してほしいと。

### 4講 講義まとめ

ジェンダー研究 -> ジェンダーの意味の変質 -> ジェンダーをめぐる論争
ジェンダー研究

- 男性と女性を超えた第三の性
- ジェンダーの超越性
- ジェンダーカテゴリーのない世界

ジェンダーをめぐる論争

- 国家はジェンダーを阻止すべきか
- ジェンダー研究はジェンダー自体をなくすものではないか

ジェンダー研究の方向性

- 多様なジェンダーがもっと生きやすい世界
- ジェンダーカテゴリーが必要な人々と、そこから離れるべき人々が共存する世界 -> トランスジェンダー、トランス女性、トランス男性
- ジェンダーカテゴリーのどこにいても全員が正当

反ジェンダーイデオロギーのジェンダー（社会全体に広がる恐怖と不安に訴え、社会の不安定性をジェンダー用語のせいに転嫁）

- 生物学的性の否定
- 自然の秩序の攪乱
- 異性愛中心の家族制度の分裂

ジェンダー研究が実際にしたこと

- 女性のリプロダクティブ・ライツと自由の強調
- 指定された生き方と愛し方の拒否
- 代替家族の可能性
- トランスジェンダーの権利擁護
- さまざまな形態の暴力への反対

## 5講 どう生きるか

講義を要約すると、生物学的性別と社会的性別であるジェンダーを区別しました。ジェンダーの概念を適切に解釈することの重要性を語りました。ジェンダーについて一部の人々が持つ不安感と恐怖についても語りました。ジェンダーが外部から来た概念であるとか、既存の生活を破壊するかもしれないということでした。

![](assets/ebs-great-minds-4-5-1.webp)

私たちにはより広い枠組みや新しいジェンダーの言葉が必要です。ジェンダーカテゴリーの内外の人々がみな全うな人生を送れるように。共に生きる世界では「私は誰か」ではなく**「私たちは誰か」**を問わなければなりません。

ジェンダーは男性を破壊していません。ただ男性優位主義に挑戦しただけです。家族制度を破壊したのではありません。さまざまな種類の家族や連帯も同様に重要な価値を持つと主張しただけです。

私たちが望むのは、いかなる差別と暴力への恐怖や検閲なしに、コミュニティの支持を受けながら生き、呼吸し、働き、愛することです。これこそが理想的に共存するコミュニティの生活です。

私たちがどのジェンダーと呼ばれ、どの性別を与えられるとき、社会は私たちに多くのことを語ります。一連の期待と理想的観念、慣習が私たちに伝えられます。

トランスジェンダーを嫌悪する者はフェミニストではありません。フェミニストならジェンダーによって誰かが差別されるべきだという考えに反対するからです。

私たち全員の課題は、複数の言語が共存する世界でより住みよい方法を見つけることです。私たちの人生の姿が他の人と違っていたり、他の人が認めてくれなくても、それ自体に価値があるという事実を知らなければなりません。どう生きるか、良い人生とは何かを考える時、一つの考えだけを持ってはいけません。世界のさまざまな個性とさまざまな言語を受け入れた時、初めて私たちは自由を見つけ、恐怖を振り払うことができます。

### 5講 講義まとめ

既存のジェンダーカテゴリーに挑戦したジェンダー研究、しかし国家がコミュニティと家族に福祉の役割を転嫁

- 反ジェンダー政策
- ジェンダー教育の廃止
- 講義、書籍、展示の検閲
- LGBTQI（レズビアン、ゲイ、バイセクシュアル、トランスジェンダー、クィア、インターセックスを指す言葉）の権利否定
- 反ジェンダー運動は進歩的価値（イスタンブール条約、国連）を毀損

反ジェンダー運動の主張

- ジェンダーは帝国主義的概念だ
- ジェンダーはキリスト教を差別する
- ジェンダーは嫌悪する自由を侵害する

どう生きるか？

- ジェンダーは世界を探究する一つの方法 -> 人生のカテゴリー、言語、自由の地平を開くジェンダー
- ジェンダーの世界を理解するために複数の言語の間を行き来しなければならない（多様性、ダイナミズム、潜在力の発見）
- 世界のさまざまな個性とさまざまな言語を受け入れる生活 -> 理想的なコミュニティの生活、共存
- ジェンダーを学ぶということ -> 文学、映画、歴史を学ぶこと
- 学生たちにも連帯が必要
