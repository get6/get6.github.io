When working with Flutter, there aren't as many reference articles available compared to React or Spring.
But there are some gems out there. Let me introduce some Flutter-related sites through these links.

## Sites

[Flutter](https://flutter.dev/): The official site you absolutely need to visit when learning Flutter.
[Dart](https://dart.dev/): You need to learn the Dart language to use Flutter. You can learn it here.
[Medium - Flutter](https://medium.com/flutter): You can find Flutter release news and change logs on Medium.

[Flutter Samples](https://flutter.github.io/samples/): Check out things built with Flutter.
[Flutter Gems](https://fluttergems.dev/): Another place to see things built with Flutter.
[IT's ALL Widgets!](https://itsallwidgets.com/): Browse apps built with Flutter.
[Flutter Awesome](https://flutterawesome.com/): Discover useful Flutter widgets.
[Flutter Tap](https://fluttertap.com/archive/): Get Flutter news via an English newsletter.
